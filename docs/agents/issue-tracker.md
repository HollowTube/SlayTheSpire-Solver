# Issue tracker: Linear

Issues and PRDs for this repo are tracked in Linear, in the **Slay The Spire Solver** project under the **Hollowtube** team (key `HOL`).

## Conventions

- All issues for this repo go into the "Slay The Spire Solver" project (Hollowtube team)
- Triage state is recorded via labels — see `triage-labels.md` for the role-to-label mapping
- Use the Linear MCP tools (`mcp__linear-server__*`) to create, read, update, and label issues, and to publish PRDs as Linear documents or issues

## When a skill says "publish to the issue tracker"

Create a new issue (or document, for PRDs) in the "Slay The Spire Solver" project via the Linear MCP tools, applying the appropriate triage label(s).

## When a skill says "fetch the relevant ticket"

Look up the issue in Linear by ID, title, or URL via the Linear MCP tools (`get_issue`, `list_issues`).
